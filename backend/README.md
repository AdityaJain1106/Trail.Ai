# â¬‡ï¸ Installation Instructions

### 1. Clone the repo

```sh
git clone https://github.com/<your-username>/trial.ai.git
cd trial.ai
```

### 2. Install Backend

```sh
cd backend
npm install
```

Create `.env`:

```
ELEVENLABS_API_KEY=your_key_here
PROJECT_ID=your_gcp_project
LOCATION=us-central1
```

Start server:

```sh
node voicechat.js
```

### 3. Install Frontend

```sh
cd voice-frontend
npm install
npm start
```

### Requirements

* Node.js 18+
* A Google Cloud project with Vertex AI enabled
* ElevenLabs API key
* Firebase project (Auth + Firestore)

Works on:

* Windows
* Mac
* Linux
* Mobile browsers
* Chrome/Safari/Edge
